# ?? Firepit Manager

Professional firepit design and configuration management system with AI-ready architecture.

## Features

### Core Capabilities
- **Dynamic Component System** - Define stones, burners, fuel systems with custom attributes
- **Design Studio** - Configure complete firepit designs
- **Rules Engine** - Business logic and compliance checking
- **Batch Compliance** - Check designs against regulations
- **Product Management** - Track progress with integrated rules engine

### AI Migration Path
- Component definitions ? Neural network features
- Design configurations ? Training examples
- Business rules ? Reinforcement learning rewards
- Performance data ? Model validation

## Tech Stack

- **Python 3.10+** - Core language
- **PySide6** - Professional Qt GUI
- **SQLite** - Local database
- **SQLAlchemy** - ORM for clean database code
- **Pydantic** - Data validation
